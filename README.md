Project Name: Ether Task Manager

Description:
This is a full-stack task management application built using Spring Boot and PostgreSQL. It allows users to manage tasks efficiently with authentication features.

Features:
- User Signup and Login (JWT Authentication)
- Create, Update, Delete Tasks
- Secure REST APIs
- Backend deployed on Railway
- PostgreSQL database integration

Tech Stack:
- Backend: Spring Boot
- Database: PostgreSQL
- Build Tool: Maven
- Deployment: Railway

Live Application:
https://web-production-f59ad.up.railway.app/

GitHub Repository:
https://github.com/mayanksharma750/ether-task-manager

How to Run Locally:
1. Clone the repository
2. Open in IntelliJ or VS Code
3. Configure PostgreSQL database
4. Run using:
   mvn spring-boot:run
